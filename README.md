# Calculator Web App
This is a simple calculator which evaluates all the expressions like real calculator.

## Features
User can calculate any of the expressions.
User can also calculate again with the result of the previous expressions.
User can clear all the expressions.
User can also back the one or more digits.

## Backend
I user the asp.net core web api

## Frontend
I user the react vite for frontend UI

## How to run this project

For backend you have just execute 2 commands
1. cd Backend
2. dotnet run

For frontend execution
1. cd Frontend
2. npm i
3. npm run dev

This is the whole documents about the projects and you can easily run this calculator in your own system.
